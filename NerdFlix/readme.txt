Para instalar as bibliotecas/depencencias do TypeScript, como desenvolvimento.

-> npm install typescript ts-node-dev -D.
-> npm install graphql graphql-scalars type-graphql

Para criar o TSConfig para inserir as configurações.

-> tsc --init.
