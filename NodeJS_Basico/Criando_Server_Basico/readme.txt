Para verificar os status HTTP acesse o https://http.cat

Metodo createServer ira criar um servidor HTTP, utilizando o NodeJS, que obrigatoriamente 
possui 2 parametros REQUISICAO e RESPOSTA.

Para executar um servidor em node, basta acessar a pasta pelo CMD e executar o comando: 

- node "aquivo.JS"
